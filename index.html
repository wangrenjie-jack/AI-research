MoE 2.0 架构

MoE 路由原理

细粒度专家路由

V4 将专家粒度从 V3 的 256 个扩展至 512 个细粒度专家，每次推理仅激活 Top-K（K=8）个专家。更细的粒度使路由决策更精准，专家 specialization 更强，在相同激活参数量下显著提升模型容量。

动态偏置算法

引入可学习的 Dynamic Bias 机制，在路由评分中叠加训练中自适应调整的偏置项。该算法有效缓解了 MoE 模型中普遍的 专家负载坍塌 问题，使 token 分配更均衡，推理吞吐量提升约 18%。

极低推理成本

1.6T 总参数中仅 49B 被激活（3.06%），单次 FLOPs 等效于一个 49B Dense 模型。相比同规模 Dense 模型，推理成本降低 ~97%，使万亿级模型在消费级 GPU 上部署成为可能。

CSA

Compressed Sparse Attention

压缩稀疏注意力（CSA）对长序列中的低信息密度 token 进行结构化压缩。通过可学习的压缩投影，将连续的 low-attention token 块压缩为单一表征，使序列有效长度缩短 4-8x，同时保留关键语义信息。

结构化 token 块压缩
可学习压缩投影矩阵
保留高信息密度锚点
Token 序列压缩示意
t1 t2 t3 t4 → c1 t5 t6 t7 → c1 c2
■ 保留/压缩锚点   ■ 低密度 token → 压缩

HCA

Heavily Compressed Attention

重度压缩注意力（HCA）在 CSA 基础上进一步对历史上下文进行层级化重度压缩。通过多级压缩管道，将远距离 token 压缩至极低维度表征，使 1M 上下文 的 KV Cache 占用降至与传统 128K 模型相当的水平。

Connectivity

mHC 流形约束超连接

Manifold-constrained Hyper-Connection — 万亿参数训练的梯度稳定器

核心原理

mHC 在残差连接中引入 流形约束的动态缩放。传统残差连接 y = x + F(x) 在深层网络中易导致梯度爆炸/消失。mHC 将其推广为：

y = α ( β · x + F ( x ))
α, β ∈ 流形约束空间，训练中自适应调整

其中 α 和 β 被约束在预定义的 黎曼流形 上，确保缩放因子始终处于数值稳定区间。这使得梯度在万亿参数的极深网络中仍能稳定传播，避免了传统 DeepNorm 等方法在超深 MoE 网络中的退化问题。

梯度稳定性

流形约束确保缩放因子有界，梯度范数在 [ε, M] 内稳定传播，消除深层 MoE 中的梯度消失/爆炸。

特征辨识度

动态缩放使不同层的特征表征在流形上保持可区分性，避免深层特征坍塌至低维子空间，提升模型表达力。

零额外推理开销

mHC 的缩放因子在推理时融合进权重矩阵，零额外 FLOPs，训练稳定性收益不带来推理成本。

Muon 优化器

Matrix Orthogonal Newton

Muon 是一种基于 矩阵正交化 的二阶近似优化器。与传统 AdamW 不同，Muon 利用参数矩阵的 谱结构 信息，通过极低成本的 QR 分解近似二阶更新方向，在保持 Adam 量级计算成本的同时获得接近二阶方法的收敛速度。

特性 AdamW Muon
收敛阶数 一阶 ~二阶
计算开销 O(n) O(n·k)
谱信息利用 ✗ ✓
训练步数节省 — ~30%

* k 为 QR 分解迭代次数，通常 k=1-3 即可，额外开销 < 5%

MTP

Multi-Token Prediction

多 Token 预测（MTP）在训练时同时预测后续 D 个 token（D=4），而非仅预测下一个。这迫使模型学习更丰富的上下文表征和更长程的依赖关系，显著提升逻辑一致性。

MTP 训练架构

逻辑一致性提升

多步预测迫使模型规划更长推理链，减少自回归误差累积，在数学推理和代码生成任务上提升显著。

推理加速

推理时可利用 MTP 头进行 speculative decoding，一次前向传播验证多个候选 token，实现 1.5-2x 推理加速。

万亿参数，消费级成本

DeepSeek-V4 通过 MoE 2.0、混合注意力、mHC 和 Muon+MTP 四大核心创新，在万亿参数规模下实现了前所未有的推理效率与训练稳定性，重新定义了大模型的成本-性能边界。
